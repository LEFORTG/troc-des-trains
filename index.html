Comment ça fonctionne ?

J'achète

  1. Je recherche
  2. Je trouve mon billet
  3. Je checkes les avis du vendeur
  4. Je paye et reçoit immédiatement mon billet

Je vends

  1. Je télécharge le billet sur l'application
  2. Je fixe le prix
  3. J'enregistre mes coordonnées bancaires
  4. Je reçoit l'argent dès que le billet est vendu

Je donne mon avis

  1. Pour rassurer les prochains acheteurs
  2. Pour avoir une bonne réputation
  3. Pour améliorer votre site
  4. Pour améliorer les échanges entres acheteurs/vendeurs

Qui payes les frais ?

  1. Prix de vente du billet 30€
  2. Le vendeur reçoit 29€
  3. L'acheteur paye 31€
  4. Troc des Trains reçoit 2€
